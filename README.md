# C++ Programlama Odevi

## NASIL KULLANILIR?

### 1. Codespace Ac

Repository sayfanizda:
- **Code** (yesil buton) -> **Codespaces** -> **Create codespace on main**

### 2. Kod Yaz

Sol tarafta `main.cpp` dosyasini acin ve kodunuzu yazin.

### 3. Gonder ve Test Et

Terminal'de (asagida) sunu yazin:

```bash
./gonder.sh
```

**VEYA** kisayol:

```bash
gonder
```

### 4. Sonucu Bekle

- 60 saniye bekleyin
- Yesil = Basarili
- Kirmizi = Hata var, mesaji okuyun

## SSS

**S: Terminal nerede?**  
C: Asagida, Ctrl + ` ile acilir

**S: Birden fazla gonderebilir miyim?**  
C: Evet, sinirsiz! Her seferinde `gonder` yazin

**S: Hata aldim, ne yapmaliyim?**  
C: Hata mesajini okuyun ve kodu duzeltin. Sonra tekrar `gonder`

**S: Test sonuclarini tekrar gormek istiyorum?**  
C: Repository -> Actions sekmesi
